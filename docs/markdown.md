# 5分钟学会 Markdown

## 标题
```
# H1 标题
## H2 标题
### H3 标题
#### H4 标题
##### H5 标题
###### H6 标题
```

## 文本格式
```
**粗体文本**
*斜体文本*
***粗体和斜体***
~~删除线~~
`行内代码`
```

## 链接和图片
```
[链接文本](https://example.com)
[带标题的链接](https://example.com "悬停标题")
![替代文本](image.jpg)
![带标题的图片](image.jpg "图片标题")
```

## 列表
```
无序列表:
- 项目 1
- 项目 2
  - 嵌套项目

有序列表:
1. 第一项
2. 第二项
3. 第三项
```

## 引用块
```
> 这是一个引用块
> 多行内容
>> 嵌套引用
```

## 代码块
````
```javascript
function hello() {
  console.log("Hello, world!");
}
```
````

## Tables
```
| 标题 1 | 标题 2 | 标题 3 |
|----------|----------|----------|
| 第一行    | 数据     | 数据     |
| 第二行    | 数据     | 数据     |
```

## 水平分割线
```
---
或
***
或
___
```

## 任务列表
```
- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个任务
```

## 转义字符
```
使用反斜杠转义: \* \_ \# \`
```

## 换行
```
在行尾添加两个空格  
来创建换行。

或者使用空行来创建新段落。
```
